# C++学习笔记
## 位左移和位右移等价于十进制的哪些操作
### 位左移
#### 二进制'<<'x位，相当于把二进制数的末尾添加x个0
##### 例子：
```bash
101010<<1=1010100
101010
1010100 // <-- 相当于多出来一个0
```
#### 位左移等价于十进制数乘以2的x次方
##### 例子：
```bash
101010<<1=1010100
42<<1=84
42
84 // <-- 相当于42*2
```
### 位右移
#### 二进制'>>'x位，相当于把二进制数的末尾删除x个0
##### 例子：
```bash
101010>>1=10101
101010
10101 // <-- 相当于多出来一个0
```
#### 位右移等价于十进制数除以2的x次方
##### 例子：
```bash
101010>>1=10101
42>>1=21
42
21 // <-- 相当于42/2    
```
