# HTML学习笔记
## 1.开头模板
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>
```
## 2.```<p>```标签
#### 作用：显示一个段落
#### 例如：
```html
<p>这是一个段落</p>
```
#### 显示为：
<p>这是一个段落</p>

## 3.```<h1>```标签
#### 作用：显示一个一级标题
#### 例如：
```html
<h1>这是一个一级标题</h1>
```
#### 显示为：
<h1>这是一个一级标题</h1>

## 4.```<DOCTYPE>```标签
#### 作用：声明文档类型为HTML5
#### 例如：
```html
<!DOCTYPE html>
```
#### 无显示
## 5.```<span>```标签
#### 作用：定义一个内联元素
#### 例如：
```html
<span>这是一个内联元素</span>
```
#### 显示为：
<span>这是一个内联元素</span>
