# HTML学习笔记
## 1.开头模板
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>
```
## 2.```<p>```标签
#### 作用：显示一个段落
#### 例如：
```html
<p>这是一个段落</p>
```
#### 显示为：
<p>这是一个段落</p>

## 3.```<h1>```标签
#### 作用：显示一个一级标题
#### 例如：
```html
<h1>这是一个一级标题</h1>
```
#### 显示为：
<h1>这是一个一级标题</h1>

## 4.```<DOCTYPE>```标签
#### 作用：声明文档类型为HTML5
#### 例如：
```html
<!DOCTYPE html>
```
#### 无显示
## 5.```<span>```标签
#### 作用：定义一个内联元素
#### 例如：
```html
<span>这是一个内联元素</span>
```
#### 显示为：
<span>这是一个内联元素</span>

## 6.```<div>```标签
#### 作用：定义一个块级元素
#### 例如：
```html
<div>这是一个块级元素</div>
```
#### 显示为：
<div>这是一个块级元素</div>

## 7.```<title>```标签
#### 作用：定义文档的标题
#### 例如：
```html
<title>这是一个标题</title>
```
#### 显示为：
这是一个标题
## 8.```<meta>```标签
#### 作用：定义文档的元数据
#### 例如：
```html
<meta charset="UTF-8">
```
#### 无显示
## 9.```<body>```标签
#### 作用：定义文档的主体
#### 例如：
```html
<body>
    <h1>这是一个一级标题</h1>
    <p>这是一个段落</p>
</body>
```
#### 显示为：
<h1>这是一个一级标题</h1>
<p>这是一个段落</p>

## 10.```<head>```标签
#### 作用：定义文档的头部
#### 例如：
```html
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>这是一个标题</title>
</head>
```
#### 无显示

## 11.```<!-- -->```标签
#### 作用：定义文档的注释
#### 例如：
```html
<!-- 这是一个注释 -->
```
#### 无显示

## 12.```<h1>,<h2>,<h3>,<h4>,<h5>,<h6>```标签
#### 作用：定义文档的标题
#### 例如：
```html
<h1>这是一个一级标题</h1>
<h2>这是一个二级标题</h2>
<h3>这是一个三级标题</h3>
<h4>这是一个四级标题</h4>
<h5>这是一个五级标题</h5>
<h6>这是一个六级标题</h6>
```
#### 显示为：
<h1>这是一个一级标题</h1>
<h2>这是一个二级标题</h2>
<h3>这是一个三级标题</h3>
<h4>这是一个四级标题</h4>
<h5>这是一个五级标题</h5>
<h6>这是一个六级标题</h6>

## 13.```<br>```标签
#### 作用：定义换行
#### 例如：
```html
这是一个段落<br>
这是第二个段落
```
#### 显示为：
这是一个段落<br>
这是第二个段落

## 14.```<wbr>```标签
#### 作用：定义软换行
#### 例如：
```html
这是一个段落<wbr>
这是第二个段落
```
#### 显示为：
这是一个段落<wbr>
这是第二个段落

## ```<hr>```标签
#### 作用：定义水平线
#### 例如：
```html
<hr>
```
#### 显示为：
<p>1</p>
<hr>
<p>2</p>

## 13.```<pre>```标签
#### 作用：定义预格式化（保留空格和换行等）文本
#### 例如：
```html
<pre>
    这是一个预格式化文本
    它会保留空格和换行
</pre>
```
#### 显示为：
<pre>
    这是一个预格式化文本
    它会保留空格和换行
</pre>
